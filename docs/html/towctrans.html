towctrans - wide-character transliteration

If wc is a wide character, the towctrans() function translates it according to the transliteration descriptor desc.
If wc is WEOF, WEOF is returned.

Variable desc must be a transliteration descriptor returned by the wctrans(3) function.

Return Value

The towctrans() function returns the translated wide character, or WEOF if wc is WEOF.

Notes

The behavior of towctrans() depends on the LC_CTYPE category of the current locale.
